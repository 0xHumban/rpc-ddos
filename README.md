# rpc-ddos
 Script to send massive request to a rpc

## How to run bot
### With executable
```
$ cd ./executable
$ ./rpc-ddos 
```

### With cargo
```
$ cargo build --release
$ ./target/release/rpc-ddos
```

## Usage
After launching the program, you will be prompted to enter: 
- HTTP RPC request URL. 
- Threads number
